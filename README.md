# Darwin Microsoft Teams Troubleshooting Lab

## Overview

This project demonstrates a common Windows 11 Help Desk troubleshooting scenario involving Microsoft Teams.

The lab walks through verifying account settings, configuring audio and video devices, reviewing notification and meeting settings, checking Windows permissions, and documenting common troubleshooting steps performed by Help Desk technicians.
